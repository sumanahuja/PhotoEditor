Description

In this stage, we will implement our first filter â€” the brightness filter.

A pixel brightness is simply the average RGB value, so:

\[brightness = \frac {(Red + Green + Blue)} 3\]

To increase/decrease it, simply add or subtract the same amount from each RGB color. We will add a slider to adjust the image brightness.

Objectives
  • Add a Slider with id slBrightness; the values may vary from -250 to 250 with step size of 10 and initial value of 0;
  • Attach an onChangeListener to this slider that will apply the brightness filter based on slider values;
  • The brightness filter should add the current value of brightness slider to each RGB value of each pixel and check whether the new value is within the correct boundaries (0, 255);
  • Every time users load a new image, you must save its default bitmap and operate on it while applying a brightness filter (not on the current bitmap, because applying filter may erase image details).

obs: You are expected to implement the filter functionality manually, by directly manipulating pixel values of your bitmap. Solutions that only affect the rendered image like using ColorFilter won't be able to pass tests.
